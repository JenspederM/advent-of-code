# Advent of Code

Running a day's solution:

```
# Run the solution for day 1
go run ./cmd/run/main.go -day 1

# Run the solution for day 2
go run ./cmd/run/main.go -day 2

...
```

Creating a new day:

```
# Create a new day 3
go run ./cmd/new/main.go -day 3
```

Running tests:

```
go test ./...
```
